Dot is not a Bird

This is a small PyGame using Python 2.7 inspired by Flappy Bird. But instead of a bird, it's a dot!

You can download this small app (6mb) which is a compressed (zip) file that you can extract. Once extracted, you will find a dot.exe and just run it to start playing.

You can use your keyboard SPACEBAR, arrow UP, or your left mouse CLICK to keep jumping through the pipes or obstacles.

I hope you will enjoy playing this game in as much as I enjoy creating it.
